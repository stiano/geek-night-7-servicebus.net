Oppsett av oppgave for .Net 
=====================

Installasjon av NServiceBus
--------------

1. Last ned NServiceBus 3.3.5
2. http://nservicebus.com/DownloadNow.aspx
3. Åpne filen og velg ja på alt

De installerte filene la seg her på min maskin: 
~\AppData\Local\Apps\NServiceBus\v3.3.5. Her finnes flere eksempelprosjekter som kan benyttes ved behov.

MSMQ - her kan du se meldingene
--------------

1. Run: compmgmt.msc
2. Åpne: Services and Applications -> Message Queuing -> Private Queues -> klient+server
3. Debug: Sett knekkpunkt i Visual Studio og stopp under kjøring.

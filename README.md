Oppsett av oppgave for .Net 
=====================

Installasjon av NServiceBus
--------------
# Last ned NServiceBus 3.3.5
# http://nservicebus.com/DownloadNow.aspx
# Åpne filen og velg ja på alt

De installerte filene la seg her på min maskin: 
~\AppData\Local\Apps\NServiceBus\v3.3.5. Her finnes flere eksempelprosjekter som kan benyttes ved behov.

MSMQ - her kan du se meldingene
--------------

# Run: compmgmt.msc
# Åpne: Services and Applications -> Message Queuing -> Private Queues -> klient+server
# Debug: Sett knekkpunkt i Visual Studio og stopp under kjøring.
